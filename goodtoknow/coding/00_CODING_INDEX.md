# 🛠️ SIPENDORA — Coding Tutorial
> Panduan praktis membangun sistem penyewaan fasilitas olahraga berbasis First Come First Served (FCFS).

Tutorial ini menjelaskan langkah-demi-langkah teknis pengembangan Sipendora dari sisi Backend (Node.js/Express) hingga Frontend (React/Vite).

---

## 🚀 Alur Pengembangan (Development Flow)

```text
STEP 1: DATABASE LAYER
01_DATABASE_DAN_KONEKSI.md
└─ Konfigurasi MySQL & Pool Connection

STEP 2: AUTHENTICATION
02_AUTHENTICATION_JWT.md
└─ Login, Register, & JWT Middleware

STEP 3: THE ENGINE (FCFS)
03_ALGORITMA_FCFS_ENGINE.md
└─ Implementasi antrean berdasarkan waktu

STEP 4: BACKEND API
04_BACKEND_API_ROUTES.md
└─ Membuat endpoint booking & pelaporan

STEP 5: FRONTEND UI
05_FRONTEND_VITE_REACT.md
└─ Struktur React & Axios Integration

STEP 6: STATE MANAGEMENT
06_STATE_MANAGEMENT_ZUSTAND.md
└─ Mengelola global state dengan Zustand
```
